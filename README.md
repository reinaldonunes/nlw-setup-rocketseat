<h1> Setup Fron-end</h1>

## 🔥 Front-end

Instale as dependências do projeto:

```
npm install
```

Rode a aplicação:

```
npm run dev
```

ou

```
npm run build
```

## 🔐 Back-end

\*\* Obs: certifique-se de ter o Node.js na versão v18.3.0 LTS instalada.

Instale as dependências

```
npm i
```

Rode o servidor:

```
cd server
npm run dev
```
